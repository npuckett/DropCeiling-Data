About this site.

Project

The project.

DropCeiling is an open-source adaptive augmented-reality installation: 12 LED panels in a Toronto storefront window, lit by a single virtual point light whose state responds to pedestrians tracked on the sidewalk outside. The system ran 24/7 for 54 days (Jan 29 – Mar 17, 2026), capturing roughly 23.3 million tracking events and changing measurably along the way.

The system is governed by three nested control loops at three timescales: reaction (modes and gestures follow who is present now, ~33 ms), anticipation (trend windows pre-position the light, seconds–hours), and self-tuning (a meta-review reshapes the twelve meta-parameters, hours–days). An interaction budget meters how fast the personality can change, keeping adaptation deliberate across the whole run.

It was developed with AI coding agents working against a shared 3D model of the site — the model didn't describe the installation, it became its brain.

Colophon

How this site was built.

One HTML page with all sections as anchors. One CSS file, ~300 lines, paper background + 1px borders + 8px radius + system fonts, with a warm-accent palette for the light. Four small vanilla-JS scripts, no dependencies, no build step. The narrative markdown is converted to HTML at build time with markdown-it-py; the static JSONs the site fetches are regenerated from the merged SQLite with IO/analysis/export_web_json.py.

The home page is patterned after the p5-phone docs — same fixed-sidebar + slide-in-drawer pattern, same eyebrow→h2→lede section rhythm, same one-CSS-file discipline. The DropCeiling palette adapts p5-phone's paper + green-accent into paper + warm-accent, the warm reserved for the light (and for the most important annotations).
